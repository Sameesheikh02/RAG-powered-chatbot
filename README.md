# 🍕 Pizza RAG Chatbot (Local AI Agent)

A conversational AI chatbot that answers questions about a pizza restaurant using Retrieval-Augmented Generation (RAG).

## 🚀 Features

- 🔍 Semantic search using vector database (Chroma)
- 🧠 Conversational memory (LangChain)
- 💬 Chat-based UI with Streamlit
- ⚡ Streaming responses (real-time token generation)
- 🏠 Runs locally using Ollama (no API cost)

## 🛠️ Tech Stack

- Python
- LangChain
- Ollama (LLM)
- ChromaDB
- Streamlit
